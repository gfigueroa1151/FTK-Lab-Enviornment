# FTK-Lab-Enviornment

FTK Imager — Disk Image Forensics Lab 

A hands-on digital forensics investigation from CyberDefenders. I played the SOC analyst investigating an employee ("Karen") suspected of illegal activity, acquiring and analyzing a disk image of her Linux workstation to reconstruct what happened.

Tools

- FTK Imager 8.2.0.59 — disk image acquisition and analysis
- Windows 10 VM — analysis environment

What I Practiced

- Disk image acquisition and file system navigation
- Locating key Linux artifacts (bash history, auth.log, web server logs, desktop files)
- Hash verification for evidence integrity (MD5 / SHA1)
- Reconstructing user activity from logs and command history
- Clear, methodical forensic report writing
